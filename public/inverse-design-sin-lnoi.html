하이브리드 SiN-on-LiNbO₃ 플랫폼의 광소자 역설계

Inverse Design of Photonic Devices in Hybrid Silicon Nitride-on-Lithium Niobate Platform · Aman Verma, Varun Raghunathan (IISc Bengaluru) · 학회 포스터 요약

← Papers summery GitHub

발행일 2026-07-01

한 줄 요약

Adjoint 기반 역설계(inverse design)로 하이브리드 SiN-on-LNOI(Silicon Nitride on Lithium Niobate on Insulator) 도파로 플랫폼에서 50:50 빔 스플리터와 모드 변환기(mode converter)를 설계·제작하고, 실측으로 검증한 연구. 제작 공정을 고려한 최적화로 25×50 µm²의 초소형 소자를 구현했다.

목표

  • 하이브리드 SiN-LNOI 플랫폼에서 컴팩트 광소자 설계.
  • Adjoint 방법 기반 위상(토폴로지) 최적화 시연.
  • 빔 스플리터와 모드 변환기를 실험적으로 검증.

플랫폼 / 재료 스택

  • SiN(290 nm) 로딩 층 + LN(500 nm) + SiO₂(2 µm) + Si 기판으로 구성된 SiN-loaded LNOI 도파로.
  • TE00 기본 모드의 76.6%가 LN 층에 집중 — LN의 전기광학 특성을 활용하기 유리.

방법

  • 하이브리드 SiN-loaded LNOI 도파로 플랫폼.
  • Tidy3D(클라우드 기반 FDTD)로 전자기 시뮬레이션.
  • 픽셀 기반 adjoint 최적화 + 제작 공정 제약(fabrication constraints) 반영.
  • 제작: 전자빔 리소그래피(EBL) + 건식 식각(dry etching).

시뮬레이션 결과

  • TE00→TE20 모드 변환기와 50:50 파워 스플리터 최적화.
  • 40~50 iteration 내 수렴 — 모드 변환 metric 약 90%, 스플리터 약 93% 달성.
  • 소자 footprint 25 × 50 µm²로 매우 컴팩트.

실험 결과

  • 1550 nm 레이저(8 mW), 파이버 결합 측정. 기존 MMI 소자와 비교.
  • SEM·광학현미경으로 50:50 스플리터·모드 변환기 제작 확인.

향후 과제

  • 모드 변환기·파워 스플리터에 더해 컴팩트 90°·U자형 벤드 설계.
  • 편광 빔 스플리터(polarization beam splitter) 통합.
  • 역설계 기반 다중포트 간섭계 네트워크 집적회로.
  • SiN-LNOI 플랫폼 위 파장 스플리터·필터.

의의 / 한계

  • 의의: Adjoint 역설계가 하이브리드 SiN-LNOI에서 컴팩트 광기능(스플리터·모드변환)을 가능케 함을 실험으로 입증. 고밀도 포토닉 집적에 유망.
  • 제작 호환성: 공정 제약을 최적화에 넣어 실제 EBL·식각으로 재현 가능한 패턴을 얻음.
  • 한계(요지): 역설계 소자의 절대 삽입 파워가 MMI보다 낮게 측정됨(손실 존재). 삽입손실·대역·수율의 추가 개선 여지.

메모 — PDL 관점

  • LNOI + SiN 로딩 하이브리드는 우리 LNOI 소자 설계와 직접 맞닿는 플랫폼 — 모드가 LN에 갇히면서도 SiN으로 패터닝하는 접근을 참고할 만함.
  • Adjoint/토폴로지 역설계 + Tidy3D 워크플로는 우리 스플리터·MMI·모드변환 설계에 도입 시 footprint를 크게 줄일 잠재력.
  • "제작 공정 제약을 최적화 목적함수에 포함"하는 방식은 설계-제작 간극을 줄이는 실용적 아이디어.
