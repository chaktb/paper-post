Modified Hermite 최적화로 만든 Si₃N₄ O-band 25µm 벤드

Si₃N₄ Bend at O-Band with 25-µm Radius and 5 mdB/90° via Modified Hermite Optimization · Donghao Li, Yi Xu, Lei Zhang (BUPT) · 학회 포스터 요약

← Papers summery GitHub

발행일 2026-07-01

한 줄 요약

Modified Hermite 곡선으로 도파로 벤드 형상을 설계해, 반경 25µm의 초소형 Si₃N₄ 90° 벤드에서 O-band(1311nm) 굽힘 손실 5.18 ± 0.38 mdB/90°를 8인치 웨이퍼 위에서 실측 시연한 연구. 데이터센터·HPC용 고집적 단거리 광연결(short-reach interconnect)을 겨냥한다.

배경

  • 데이터센터·HPC의 트래픽 증가로 소형·저전력 단거리 광연결 수요 확대.
  • O-band(저분산) + 실리콘나이트라이드(Si₃N₄) 조합이 이런 링크에 이상적.
  • 고집적화의 관건은 굽힘 손실(bending loss) — 반경을 줄이면 손실이 커지는 문제.
  • 이 연구는 Modified Hermite 설계로 25µm 반경에서도 저손실을 달성.

원리 / 설계

  • Modified Hermite 벤드: 두 개의 Hermite 곡선이 도파로 입출력점(P₃, P₄)에서 시작·종료하며, 내측/외측 경계를 이룬다. 네 개의 고정점은 굽힘 반경 R로 결정된다.
  • 곡선은 매개변수 t(0→1)로 표현되며, 45°에서 도파로 폭이 최대가 되는 구조적 대칭성을 가진다. 이 대칭성 덕분에 계수가 a₁=a₃, a₂=a₄로 묶여, 고정 반경에서 손실 최소화가 a₁·a₂ 두 파라미터 최적화로 단순화된다.
  • 시뮬레이션상 초저 전파 손실, 우수한 파장 무의존성, 높은 공정 오차 허용도를 보인다.

최적화 방법

  • 3D-FDTD 솔버로 준기본 TE₀ 모드 해석.
  • PSO(입자 군집 최적화, Particle Swarm Optimization)로 파라미터 탐색 → 최적값 a₁ = 44.0, a₂ = 50.7.
  • 최적 파라미터로 TE₀·TE₁ 굽힘 손실을 3D-FDTD로 재시뮬레이션 → 1311nm에서 TE₀ 손실 약 4 mdB/90°.

제작 / 실험 결과

  • 8인치 실리콘 웨이퍼 위에 180nm 포토리소그래피로 제작.
  • 광 스택: LPCVD로 성장한 300nm SiN 코어, 하부 8µm BOX, 상부 3.55µm 실리카 클래딩.
  • 25µm 벤드의 cut-back 측정: 손실–벤드 개수의 선형 상관 → 피팅 굽힘 손실 5.42 mdB/90°.
  • 웨이퍼 레벨 균일도 검증: 벤드 손실 4.19 ~ 5.87 mdB/90°, 평균 5.18 ± 0.38 mdB/90° — 공정 균일성 우수.

의의 / 한계

  • 의의: 형상 설계(Modified Hermite)만으로 25µm 초소형 반경에서 저손실을 확보 — 고집적 O-band 포토닉 회로에 유리. 웨이퍼 레벨 균일도까지 입증.
  • 대칭성 활용: 2-파라미터(a₁, a₂) 최적화로 설계 공간을 크게 축소해 PSO 탐색을 효율화.
  • 한계(요지): TE₀ 기준 결과가 중심이며, 고차 모드·편광·대역 전반의 특성은 추가 검증 여지가 있음(포스터 범위 밖).

메모 — PDL 관점

  • 도파로 벤드 형상 최적화로 굽힘 손실을 줄이는 접근은 우리 도파로/IOC 레이아웃 설계에 직접 참고 가능(오일러 벤드 대안).
  • "구조적 대칭성 → 파라미터 결합 → 최적화 차원 축소" 아이디어는 다른 소자 설계에도 응용 가치가 있음.
  • LPCVD SiN + 실리카 클래딩 스택과 cut-back 손실 측정 절차는 우리 공정·평가 흐름과 비교해볼 만함.
