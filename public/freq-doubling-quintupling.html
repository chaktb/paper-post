위상→세기 변조를 이용한 주파수 2배~5배 체배

Frequency Doubling to Quintupling by Using Phase to Intensity Modulation · Sangjoon Park, Kyoungsik Yu (KAIST EE) · 학회 포스터 요약

← Projects GitHub

발행일 2026-07-01

한 줄 요약

위상 변조기(phase modulator)와 단일모드 광섬유의 분산(dispersion)을 이용해 위상 변조를 세기 변조로 바꾸고, 이를 통해 밀리미터파 대역 RF 신호의 2배·3배·4배·5배 주파수 체배를 하나의 간단한 구성으로 실험 시연한 마이크로파 포토닉스(MWP) 연구.

배경 — 마이크로파 포토닉스(MWP)

  • MWP의 장점: 초광대역, 저손실 광 분배, 전자기 간섭에 강한 내성 — mm파 신호 생성, RoF(Radio-over-Fiber), 광 빔포밍 네트워크, 포토닉 RF 필터에 활용.
  • 최근 MWP 시스템은 60 GHz 이상의 캐리어 주파수에 집중.
  • 기존 포토닉 RF 주파수 체배 방식: 위상잠금루프(PLL), 주입잠금(injection locking), 외부 변조(external modulation).
  • 이 중 외부 변조가 구성이 단순하고 재현성이 높으며 표준 MWP와 호환되어 가장 실용적 — 본 연구가 채택한 접근.

방법 / 이론

  • 위상→세기 변조 개념: CW 레이저 → 위상 변조기 → 광섬유(분산 매질) → 광검출기(PD). 위상 변조가 분산을 거치며 세기 변조로 변환됨.
  • 이상적 위상→세기 변조에서 k차 고조파 진폭은 대략
    |i_k(ω_m, β₂L)| ∝ |J_k( 2m·sin(½ β₂ L ω_m²) )|
    형태로, 누적 분산에 대해 주기적·대칭적으로 변한다. → 높은 변조 주파수에서 고차 체배 최적화가 가능.
  • 성능 지표(FOM) 정의: 원하는 체배 톤이 다른 고조파 대비 얼마나 지배적인지 정량화 —
    ⟨i_n/m⟩² / Σ_k ( (i_k/m)² − (i_n/m)² )
  • FOM으로 찾은 최적 변조 지수(modulation index): 2배 ≈ 0.7π rad, 3배 ≈ 0.61π rad, 4배·5배는 π rad 부근.

실험 구성

  • 파장 λ₁ = 1550 nm 튜너블 레이저(TLS) → 편광 조절기 → LiNbO₃ 위상 변조기 → G.652D 단일모드 광섬유(SMF) → 광검출기(PD) → 샘플링 오실로스코프.
  • 순수 정현파 RF와 충분한 위상 변조를 얻기 위해 RF 필터 뒤에 RF 증폭기 사용.

결과

이론에서 유도한 FOM을 적용해 최적 조건을 찾고, 2배부터 5배까지의 주파수 체배를 실험적으로 시연. 대역 제한과 HPF 억압을 적용해 불요 성분을 줄인 뒤에도 목표 RF 파워를 −25 dBm 이상으로 유지.

의의 / 한계

  • 의의: 단일 위상 변조기 + 광섬유 분산이라는 간단한 구성으로 2~5배 체배를 모두 커버 — 저삽입손실, 단순 구성, 목표 주파수에서 높은 출력 파워.
  • 확장성: 변조기·광검출기 대역폭이 향상되면 더 높은 주파수로 확장 가능할 것으로 기대.
  • 한계(요지): 고차 체배 시 불요 고조파 억압을 위한 대역 제한·HPF가 필요하며, 실용 파워 확보를 위해 RF 증폭이 요구됨.

메모 — PDL 관점

  • LiNbO₃ 위상 변조기 + SMF 분산 기반 접근으로, 우리 phase modulator / IOC 프로젝트와 소자·플랫폼 측면에서 연관성이 큼.
  • "위상 변조 → 분산 → 세기 변조" 변환의 주기적·대칭적 특성(β₂L ω_m² 항)은 도파로 분산 설계 시 참고할 만함.
  • FOM 기반 변조 지수 최적화(2배 0.7π, 3배 0.61π, 4·5배 ≈ π)는 유사 체배 실험 설계 시 초기 동작점 참고치로 활용 가능.
