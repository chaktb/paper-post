2. ULL (Ultra Low Loss) waveguide

PECVD 박막 증착 조건 확인 · 담당: 정지호

← Projects GitHub

개요

  • O₂, N₂, Ar 분위기 500도에서 결과 비교
  • 저온(500도) 열처리한 도파로 특성 비교 분석
  • 박막 스트레스 분석 의뢰
    • KRISS Nano indentation 장비를 이용하여 Young's Modulus와 hardness 측정
    • 생기원에 Residual stress 장비로 분석 의뢰 — 박막 증착 조건별 확인 필요
  • 증착 조건에서 조성별 특성 ULL(Ultra Low Loss) 특성 확인 (현 0.02 dB/cm → 0.0002 dB/cm 개선 필요) — PDL 확인 필수
  • 기타 Si(100) 기판 위에 under clad – core – over clad 도파로 구현 실험 (고온 1200도 및 저온 450도 열처리 포함)
